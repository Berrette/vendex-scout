# Вендэкс Scout

Локальная PWA-CRM для клиентов, КП, напоминаний, договоренностей, фото точки, воронки и карты.

## Быстрый запуск

Откройте `start-local.cmd`. В окне появится локальный адрес:

```text
http://127.0.0.1:5193
```

## Для iPhone

Для установки как PWA страницу нужно отдавать через HTTPS. После публикации откройте адрес на iPhone и выберите Share -> Add to Home Screen.

## Публикация через GitHub Pages

1. Создайте новый публичный репозиторий на GitHub, например `vendex-scout`.
2. Загрузите в него содержимое этой папки.
3. Откройте Settings -> Pages.
4. В разделе Build and deployment выберите Source: Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Сохраните. Через минуту появится ссылка вида `https://username.github.io/vendex-scout/`.
7. Откройте эту ссылку на iPhone в Safari и добавьте на экран домой.

## Данные

Карточки клиентов хранятся локально в браузере. В приложении есть экспорт и импорт JSON для резервной копии.

## Что внутри

- экран "Сегодня" со сводкой действий;
- воронка продаж по этапам;
- карточка клиента с горячестью, следующим действием и КП;
- таймлайн звонков, сообщений, встреч и договоренностей;
- разведка точки: тип, потенциал, трафик, конкуренты;
- фото точки с хранением внутри локальной базы;
- карта клиентов на OpenStreetMap;
- шаблоны сообщений без ИИ;
- напоминание о резервной копии.
